# Zettelkasten Principles

The Zettelkasten (German for "slip box") is a system of note-taking and personal knowledge management.

## Key Principles
1. **Atoms**: Each note should contain one idea.
2. **Autonomy**: Each note must be self-contained and understandable on its own.
3. **Connectivity**: Always link notes to others. A note is only as valuable as its connections.
4. **No Hierarchy**: Avoid rigid folder structures. Let the structure emerge through links.
5. **Context**: Explain why you are linking notes.
